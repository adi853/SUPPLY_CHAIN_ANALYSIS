## üöÄ Key Business Insights
After analyzing the dataset, the following critical insights were identified:
- **Operational Risk:** A high **54.83% Late Delivery Rate** was detected, indicating a need for logistics optimization.
- **Top Markets:** **EE.UU (USA)** and **Puerto Rico** are the leading contributors to total profit.
- **Profitability Gap:** Using Scatter Plot analysis, it was observed that several high-sales categories have very low-profit margins, suggesting a need for pricing strategy adjustments.
